- COURSES and REFLECTION -
I have enjoyed one semester in the Information Technology program at the NSCC. The first semester of this program is comprised of 6 courses, each unique to one another. Course content ranged from coding in web and programming languages (HTML5, CSS3, VB.NET), to Network Simulation with Packet Tracer, to Database Development and Administration, to Technical Communications and Technical Foundations.
  • WEBSITE DEVELOPMENT
  • PROGRAMMING
  • NETWORKING
  • DATABASE
  • TECHNICAL COMMUNICATIONS
  • TECHNICAL FOUNDATIONS
WEBSITE DEVELOPMENT 1000 Website Development involved two main components: HTML5 and CSS3. We began with the basic structure of a webpage and the many elements and attributes that make up its foundation. We then progressed to CSS3 and how we could link that to our HTML5 page to customize the layout of the webpage. We started off simple, changing the background colors, font families and we gradually progressed to forms, fieldsets, tables, and links.

Software used: Notepad++
Languages used: HTML5, CSS3
PROGRAMMING 1100 Programming was a fast-paced course that focused on the creation of small, functioning programs. We started with, and built on, the simplest of concepts in the programming language. Beginning with declaring variables to if/else statements and gradually we began to work with For/Do While/Do Until loops and 2-Dimensional Arrays. While learning these concepts and their functionalities we created small, working application such as a subnet calculator, a version of the dice game Yahtzee, filling a magic square and a small quiz that had the ability to load and save.

Software used: Microsoft Visual Studio
Languages used: vb.net
NETWORKING 1100 Networking highlighted the various aspects of networks, both small and large, the theory behind the OSI Model (and all the layers), and Packet Tracer simulation. The beginning of the course was theory-content heavy as we were taught about the process of computer communication on the same network. We graduated to full network simulation in Packet Tracer where were able to configure all devices on a LAN or WAN network so they could successfully communicate with one another. We were exposed to the key components on a network (switches, routers, servers) and what role they play in the communication process.

Software used: Packet Tracer
DATABASE 1100 Database 1100 introduced the fundamentals of what a database system is. We began at the basics with Entity Relationship Diagrams (ERD) and Data Dictionaries to ensure we had a solid foundation to build on. We soon evolved into the creation of SQL statements and Queries that resulted in exposure to the functionality of a Database. To end the course, we built a summary report that computed relevant sales totals grouped by each employee. Furthermore, we brought all this together with a navigation structure that linked all forms and reports relating to that ERD.

Software used: Microsoft Visio, Microsft Access
Languages used: SQL
TECHNICAL COMMUNICATIONS 1100 Technical Communications focused on the professional inner workings of the Technology industry. We were taught various topics that ranged from the creation of a resume and a cover letter to a short report and presentation on a technology subject to mock interviews and proper grammar.
TECHNICAL FOUNDATIONS 1000 Technical Foundations brought to light many unknown, or fogotten, abilities of the many common software applications in today's industry. We completed assignments that would expose new characteristics of these platforms and how each could be used to its full potetial.

Software used: Microsoft Word, Microsoft Excel, Microsoft Powerpoint, Microsoft OneNote
